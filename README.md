## 🙏 BLESS

A custom animated splash screen for **KDE Plasma 6** featuring **KAngel** from **NEEDY GIRL OVERDOSE**.

Built entirely with **Qt6** and **QML**, this splash screen integrates natively with Plasma's splash system to replace the default startup animation with a brighter, more blessed boot experience.

Whether you're starting your day or rebooting after another all-nighter, let **Internet Angel** welcome you back.

## 📦 Installation

Since this package is not yet available on the KDE Store, you'll need to install it manually.

### Clone the repository

```bash
git clone https://github.com/shandikadav/kangel-splash.git
cd kangel-splash
```

### Install locally

```bash
mkdir -p ~/.local/share/plasma/look-and-feel/com.shandikadav.kangelsplash

cp -r * ~/.local/share/plasma/look-and-feel/com.shandikadav.kangelsplash/
```

### Enable the splash screen

Open:

**System Settings → Colors & Themes → Splash Screen**

Select **KAngel**, then click **Apply**.
